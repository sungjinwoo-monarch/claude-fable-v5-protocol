This repository contains the complete, single-file source code for the **Claude Fable 5 Mastery Guide**. A typography-focused, high-performance web interface designed to provide a deep-dive into the behavioral patterns and scaffolding techniques of the Fable 5 and Mythos 5 models.

## ⛩️ Design Philosophy
The interface is built using **Neo-Japanese Graphic Design** principles:
- **Asymmetric Grids:** Fluid layouts optimized for Desktop and Mobile.
- **Fluid Typography:** Utilizing CSS `clamp()` functions for seamless scaling.
- **Wabi-Sabi Aesthetic:** Minimalist code, high-impact visuals.

## 🧠 What's Inside
- **Long-Horizon Autonomy:** Strategies for multi-day AI runs.
- **The Send-to-User Tool:** JSON schemas for asynchronous agent communication.
- **Behavioral Steering:** Patterns to eliminate "AI-smell" and summaries.
- **Memory Systems:** Instructions for building persistent Markdown-based AI memory.

## 🚀 Deployment
To host this guide yourself:
1. Fork this repository.
2. Ensure your main file is named `index.html`.
3. Go to **Settings > Pages**.
4. Set the Source to `Deploy from a branch` and select `main`.
5. Your "God-tier" guide will be live at `https://<your-username>.github.io/claude-fable-v5-protocol/`.
